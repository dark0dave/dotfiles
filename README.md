# myDotFiles

My Dot files, for zsh. tmux and vim.
This repo if for macos and linux only.

## Setup

Ensure, you have installed:
* zsh
* curl
* direnv
* jenv
* vim,
* python
* tmux

Clone with:
```
git clone https://gitlab.com:dark0dave/myDotFiles ${HOME}/.myDotFiles
```

To start using my dotfiles, run:
```
./install.sh
```
